## Problem 題目

Bert 想要一個 n 層的三角形，第 i 層就要有 i 個 " * "，請你寫個程式幫幫可憐的 Bert

* 輸入說明：單筆輸入，輸入只有一個整數 n (1 <= n <= 100)
* 輸出說明：輸出整個三角形

## Sample 測資範例

### #1

|Input:
|:-|
|3

|Output:
|:-|
| *<br> * *<br> * * *<br>
